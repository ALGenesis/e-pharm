##e-pharm

Afin de rajouter de nouvelles pages HTML, veuillez :

- copier la page template.html
- Insérer le contenu relatif à la page dans le main
- Copier le fichier template.css 
- Styliser les éléments du main sans affecter les autres éléments du body en priviligeant les class et id
- Rajouter le code pour le responsive dans la page template.css
- N'oubliez pas de renommer vos fichiers html et css avec le meme nom
- Tester, puis déployer
